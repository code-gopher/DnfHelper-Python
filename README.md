## 如何启动

1. 下载并安装 Python 3.10.8  
   👉 [点我下载](https://www.python.org/ftp/python/3.10.8/python-3.10.8-amd64.exe)
2. 克隆仓库  
   ```bash
   git clone https://ghproxy.com/https://github.com/SunnyEmbrace/DnfHelper-Python.git
   ```
3. 安装依赖  
   ```bash
   pip install -i https://mirrors.cloud.tencent.com/pypi/simple -r requirements.txt
   ```
4. 将根目录 `static` 下的 `helper.ini` 和 `main.py` 复制到同一级目录
5. 以 **管理员方式** 运行：  
   ```bash
   python main.py
   ```
   然后登录游戏
6. 功能热键说明：
   - `F1`：全屏打怪  
   - `End`：自动刷图
